## Getting started


&nbsp;
### Clone this repository
```sh
git clone https://github.com/rafatrigo/github-explorer.git github_explorer
```

&nbsp;
### Install dependencies
```sh
#go to the project folder
cd github_explorer

#install the dependencies
yarn
```

&nbsp;
### Start the project
```sh
yarn start
```
